This is an eclipse update site for an m2e-connector for the yuicompressor-maven-plugin.

Usage Instructions
==================
1. In your Eclipse Workspace, Navigate to Help > Install New Software
2. Paste the URL https://coderplus.github.io/m2e-connector-yuicompressor-maven-plugin into the Work with textbox, click Add
3. Once added, you can install the M2E connector by following the instructions on screen
